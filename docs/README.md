## Key Principles

- **Performance First**: OnPush change detection, signals, optimized rendering
- **Accessibility**: WCAG 2.1 AA compliant out of the box
- **Theming**: CSS variables everywhere, no hardcoded values
- **Forms**: Seamless Reactive Forms integration via ControlValueAccessor
- **Flexibility**: Multiple API patterns for different use cases
